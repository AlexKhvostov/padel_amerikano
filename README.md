# Падел Американо

Веб-сервис для проведения любительских турниров по системе «Американо» в падел-теннисе.

## Стек

- Frontend: Vanilla JS, HTML5, CSS3 (mobile-first)
- Backend: PHP
- База данных: MySQL

## Конфигурация

1. Скопируйте `config/.env.example` в `.env` в корне проекта
2. Заполните реквизиты подключения к MySQL и другие параметры
